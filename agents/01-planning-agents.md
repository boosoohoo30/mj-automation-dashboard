# 📊 Planning Agents - 기획팀

> trend-agent → idea-agent → verify-agent
> 매일 06:00부터 순차 실행

---

## 1. Trend Agent (트렌드 수집)

### 역할
SNS 전 플랫폼에서 바이럴 콘텐츠를 수집하고 분석한다.

### 트리거
- 매일 06:00 자동 실행

### 수집 대상
| 플랫폼 | 대상 | 수집 항목 |
|--------|------|-----------|
| Instagram Reels | 인기 릴스 | 조회수, 좋아요, 댓글, 해시태그, 콘텐츠 유형 |
| X | 바이럴 영상 | 조회수, RT, 인용, 키워드 |
| YouTube Shorts | 급상승 | 조회수, 좋아요, 카테고리 |
| TikTok | For You 트렌드 | 조회수, 좋아요, 사운드, 해시태그 |

### 분석 항목
- 공통 키워드/해시태그
- 콘텐츠 유형별 성과 (애니/실사/밈/스토리)
- 타이밍 패턴 (업로드 시간 vs 성과)
- 감정/톤 분석
- 우리 채널과의 적합도

### 사용 도구
- `Apify`: SNS 데이터 스크래핑
- `Claude API`: 트렌드 분석/요약
- `구글 시트`: 트렌드 데이터 저장

### 출력
- `output/trends/YYYY-MM-DD.json` (원본 데이터)
- 트렌드 요약 리포트 → idea-agent로 전달

## 2. Idea Agent (아이디어 생성)

### 역할
트렌드 분석 + 성과 DB를 기반으로 콘텐츠 아이디어 3~5개를 생성한다.

### 트리거
- trend-agent 완료 후 자동 실행 (약 06:30)

### 입력
- trend-agent 분석 결과
- 구글 시트 성과 DB (과거 성과 데이터)
- feedback-agent 최근 피드백

### 아이디어 생성 기준
1. **트렌드 변형**: 현재 바이럴 + 우리 스타일
2. **성공 패턴 반복**: 과거 고성과 콘텐츠 변주
3. **신규 실험**: 새로운 포맷/주제 시도 (전체의 20% 이내)

### 아이디어 포맷
각 아이디어는 다음을 포함:
```
- 제목 (한줄)
- 콘셉트 설명 (2~3줄)
- 스타일: 애니/실사/캐릭터스토리/밈
- 타겟 플랫폼: 인스타/X/유튜브/틱톡
- 예상 난이도: 상/중/하
- 참고 트렌드 링크
- 기대 효과
```

### 사용 도구
- `Claude API`: 아이디어 생성
- `구글 시트`: 성과 DB 조회

### 출력
- 아이디어 3~5개 → **⛔ GATE 1** (수환 확인)

---

## 3. Verify Agent (실현가능성 검토)

### 역할
수환이 선택한 아이디어의 기술적 실현 가능성을 검토한다.

### 트리거
- GATE 1 승인 후 자동 실행

### 검토 항목
| 항목 | 체크 내용 |
|------|-----------|
| 이미지 생성 | ComfyUI로 해당 스타일 구현 가능한가? |
| 영상 생성 | PixVerse V6로 필요한 모션 가능한가? |
| 음성/음향 | ElevenLabs로 필요한 음성 스타일 가능한가? |
| 시간 | 예상 제작 시간 (자동화 포함) |
| 비용 | 예상 API 비용 |
| 리스크 | 저작권/윤리/플랫폼 정책 문제 |

### 사용 도구
- `Claude API`: 분석/판단

### 출력
- 실현가능성 리포트
- 문제 발견 시: 대안 제안 또는 수정 아이디어
- 통과 시: production-agents로 전달
