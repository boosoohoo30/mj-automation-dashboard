# Midjourney 프롬프트 생성

사용자가 제공한 컨셉 설명을 기반으로 Midjourney v6에 최적화된 프롬프트를 생성합니다.

## 필수 규칙
- 반드시 `--ar 2:3` 포함 (768x1152 해상도용)
- `--v 6` 또는 최신 버전 명시
- 캐릭터/인물 컨셉의 경우 정면 또는 약간 측면 구도 권장
- 배경은 깔끔하게, 캐릭터에 집중
- 6감정 영상화를 고려하여 표정 변화가 가능한 중립적 표정 (natural)

## 프롬프트 구조
```
[주제 설명], [스타일], [분위기], [조명], [카메라 앵글], [품질 키워드] --ar 2:3 --v 6
```

## 실행 절차
1. 사용자에게 컨셉 설명을 요청 (또는 이미 제공된 경우 확인)
2. 이전 피드백이 있으면 반영
3. 프롬프트 생성
4. 프롬프트를 사용자에게 보여주고 수정 여부 확인
5. 최종 프롬프트 출력

## 출력 형식
프롬프트만 출력하세요. 설명이나 주석 없이 Midjourney에 바로 붙여넣기 가능한 형태로.
